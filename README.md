# Open ICPC Trainings at KIT

At KIT we offer a weakly open ICPC training, where everyone can participate. Information about when this training is happening can be found on our [discord server](https://icpc.iti.kit.edu/post/discord/).

This repository contains material used during these trainings. Starting from the summer-semester 2026, short presentations on ICPC related topics are held at the start of each training. Presentation notes for these presentations are given here.
